# Upload and deploy

1. Unzip this package.
2. Open the `space-coast-stories` folder and upload its contents to the root of the GitHub repository, replacing matching files.
3. Commit the changes. Netlify should deploy automatically with Hugo 0.164.0 from `netlify.toml`.
4. Open the two new articles and a few category/series pages after the deploy.
5. Start a new Ubersuggest site audit. Old warnings can remain visible until the site is crawled again.
